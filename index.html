Introduction

Bony is pretty modern and fully responsive. As you might see, it scales up pretty well and feels clean. I think it is great for typography-oriented content. This includes manuals, for example.
This project originates from Dude's Skeleton CSS, quite some time ago. It has been left unmaintained, so Second Dude picked up and made his Barebones fork. I forked that one and included Third Dude's commits about :root element and and another one, for a total of two.

I mean, look at the typograpy: It gives you a feeling of peace.

Getting started

Here I am teaching you how to create a simple page with Bony. You can follow this short tutorial step-to-step or download the example files, the download link is at the end of the page.

First, let's populate a html page with the basics:

<html lang="en">

<head>
	<title>Your awesome page</title>
	<link type="text/css" href="css/bony.min.css" rel="stylesheet">
	<link type="text/css" href="css/normalize.css" rel="stylesheet">
	<link type="text/css" href="custom.css" rel="stylesheet">
	<link href="https://fonts.googleapis.com/css?family=Raleway:400,300,600" rel="stylesheet" type="text/css">
	
	<meta charset="utf-8">
	<meta name="viewport" content="width=device-width, initial-scale=1">
</head>

<body>
</body>

</html>
