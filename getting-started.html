Basic structure

First, let's populate a blank html page with the basics. It's important to add the meta tag in order to have your page to scale on mobile devices and different screen sizes.

<html lang="en">

<head>
	<title>Your awesome page</title>
	<link type="text/css" href="css/bony.min.css" rel="stylesheet">
	<link type="text/css" href="css/normalize.css" rel="stylesheet">
	<link type="text/css" href="custom.css" rel="stylesheet">
	<link href="https://fonts.googleapis.com/css?family=Raleway:400,300,600" rel="stylesheet" type="text/css">
	
	<meta charset="utf-8">
	<meta name="viewport" content="width=device-width, initial-scale=1">
</head>

<body>
</body>

</html>

The grid system

Full-sized section

Now we want to add some simple content, inside the body tag.

<div class="grid-container full">
	<div class="row">
		<h2>Bony CSS</h2>
		<p>This is my first Bony page!</p>
	</div>
</div>

First div: grid-container class allows us to make use of the grid system, while full class makes its child to be shown on a single -wide- column.

Second div contains our actual content. It is part of a row class, which is used to format text and goodies in a suitable way for text content.

Subdivisions

We have quite a few classes we can can choose to add after the grid-container, other than full:

  • Halves: Splits the page in two columns;
  • Thirds: Breaks the content in three columns;
  • Quarters: Four whole columns for our page;
  • Fifths: Allows us to have five columns.

All of these classes are responsive, columns will extend on narrower devices, and will be shown as columns on big-enough screens only. Note that columns are by default of equal width: You might need to add a custom CSS class to one of your div if you wish to obtain a 1/3 and 2/3 division.
